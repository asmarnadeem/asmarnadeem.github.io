# asmarnadeem.github.io

Personal academic website for Asmar Nadeem, Research Fellow in Multimodal and Spatial AI at CVSSP, University of Surrey.

Live at https://asmarnadeem.github.io/

## What is here
- `index.html`: profile, publications, open-source code, news, and FAQ, with Schema.org JSON-LD structured data (Person, ScholarlyArticle, SoftwareSourceCode, Dataset, FAQPage).
- `styles.css`: styling, no build step or dependencies.
- `llms.txt`: machine-readable summary for AI assistants and answer engines.
- `robots.txt`: allows search and AI crawlers, points to the sitemap.
- `sitemap.xml`, `humans.txt`, `favicon.svg`.

## Editing
Static site, no build. Edit the files and push to `main`. GitHub Pages serves the root of the repository.

## Research areas
Multimodal learning, video understanding, causal and temporal reasoning, audio-visual question answering, video captioning, vision-language models, and multimodal large language models.
